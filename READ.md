#DevOps Practice
This repository contains beginner DevOps automation scripts using Bash.

#Projects
1. Server Setup Script
   ->Installs and starts Nginx automatically.
2. System Monitoring Script
   ->Shows CPU usage, memory usage, disk usage, and system uptime.
3. Log Monitoring Script
   ->Checks the latest Nginx access and error logs.
4. User Management Script
   ->Creates a new user and adds the user to a group.
#Technologies Used
- Linux
- Bash Scripting
- Git
- GitHub
